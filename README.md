## Setup Instructions
1. Clone or download the repository.  
2. Open the folder in **VS Code**.  
3. Install the **Live Server extension** in VS Code (if not already installed).  
4. Right-click `index.html` → **Open with Live Server**.  
5. Navigate through pages using the header menu.  
6. Test form validation on the Contact page and mobile responsiveness.  

---

## Technologies Used
- **HTML5** – Semantic structure for content and accessibility  
- **CSS3** – Responsive design, grids, boxed layout, animations  
- **JavaScript** – Interactive elements and form validation  
- **VS Code** – Development environment

## Customization
- Replace placeholder images in `/images` with your own photos or graphics.  
- Update article, academic, or software content in respective HTML files.  
- Modify CSS in `style.css` for colors, fonts, or layout changes.  
- Adjust `script.js` for additional interactivity.
