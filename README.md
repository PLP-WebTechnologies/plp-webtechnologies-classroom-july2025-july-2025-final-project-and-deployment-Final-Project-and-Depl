## 🎯 Project Purpose  
M‑PesaAssist is a chatbot system aimed at providing **customer support**, **transaction help**, and **financial guidance** to M‑Pesa users. It assists users with FAQs, transaction steps, spending insights, security advice, and more — all through an intuitive conversational interface.

## 🧭 Key Features / Use Cases  
- **Transaction Support**: Guidance for send, withdraw, status checks, reversals  
- **Spending Insights / Budget Tips**: Suggest spending patterns, budgeting, alerts  
- **FAQ & Instant Help**: Quick answers to common M‑Pesa questions  
- **Security Alerts**: Advice on fraud, suspicious transactions, account safety

## 🛠 Setup & Running Locally  
1. Clone or download this repository.  
2. In root folder, open `index.html` via browser (or use a local server for proper routing).  
3. Navigate between pages to test links.  
4. On **Contact** page, test submitting — you’ll get validation responses.  
5. Later, you can hook contact form to a backend (Node.js, serverless, email API) or a form service (e.g. Formspree, Netlify Forms).
