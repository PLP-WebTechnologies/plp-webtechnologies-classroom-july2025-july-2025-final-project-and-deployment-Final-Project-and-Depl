My Portfolio Website

Project Purpose

This is a multi-page portfolio website showcasing my skills, projects, and contact information. The site is designed to be responsive, interactive, and visually appealing, demonstrating proficiency in HTML5, CSS, and JavaScript.

Structure

The project is organized as follows:





/index.html: Home page with a hero section and call-to-action.



/about.html: About page with personal information.



/projects.html: Projects page showcasing sample projects with animations.



/contact.html: Contact page with a validated form.



/css/styles.css: Stylesheet for responsive design and animations.



/js/scripts.js: JavaScript for interactivity (menu toggle, form validation, scroll animations).



/images/: Placeholder for images (not included in this submission).

Features





Responsive Design: Mobile-friendly layout with a collapsible navigation menu.



CSS Animations: Fade-in animations on project cards and hover effects on buttons.



JavaScript Interactivity: Mobile menu toggle, form validation, and scroll-based animations.



Semantic HTML: Uses HTML5 elements like <header>, <nav>, <main>, and <footer>.



Organized Code: Modular CSS and JavaScript with clear comments and folder structure.

Deployment Instructions

To deploy this website using GitHub Pages:





Push the repository to GitHub.



Go to the repository settings, navigate to the "Pages" section.



Set the source to the main branch and root directory.



Save and wait for the site to build. The live URL will be provided (e.g., https://username.github.io/repo-name).

Alternatively, for Netlify or Vercel:





Connect the repository to Netlify/Vercel via their dashboard.



Set the build command to none (static site) and publish directory to /.



Deploy and access the live URL provided.
